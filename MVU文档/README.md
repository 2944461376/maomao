## 🚀 快速开始

### 如果你是新手
1. 先阅读 [MVU基础教程](./1-MVU基础教程.md)
2. 实际操作一遍安装和配置
3. 用简单的例子测试

### 如果你想用高级功能
1. 确保已掌握基础教程内容
2. 阅读 [MVU Beta 完整指南](./2-MVUBeta完整指南.md)
3. 了解 `$meta` 配置和新命令

### 如果你要写复杂卡
1. 学习 [EJS 实战指南](./3-EJS实战指南.md)
2. 掌握世界书动态调用
3. 实现多阶段人设系统

### 如果你想优化 AI 表现
1. 使用 [AI 提示词模板](./4-AI提示词模板.md)
2. 根据你的变量系统调整 COT 流程
3. 测试和优化提示词
